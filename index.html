About PixelScript

PixelScript is an extremely minimal language, developed to be a first step in tokenized imagery(or representative rendering.)

The PixelScript held within each token is a set of instructions for any javascript enabled engine, to render the image. This includes most browsers and game engines.

Current NFT architecture leads to a token, which leads to an image download, which the engine has to download, and then render.

Not only is PixelScript much smaller than similiar pixel artwork, it's removal of multiple steps between NFT access and usage, will undoubtably promote interoptability.
